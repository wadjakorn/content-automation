# 06 — Build Phases

- **Phase 0 — consistency engine (คุ้มสุด):** Idea backlog + Planning + script/storyboard + YouTube auto-schedule อย่างเดียว (`publishAt`). ตัวขับรายได้จริง = ความสม่ำเสมอ
- **Phase 1 — feedback loop:** YouTube comments + Analytics → KB → ป้อน backlog
- **Phase 2 — multi-platform:** IG (app review) + TikTok (draft-first ระหว่างรอ audit); derivative assets; localization fan-out (auto-caption + translation + multi-lang, ดู `docs/09-localization.md`)
- **Phase 3 — scale:** Temporal, sentiment/topic clustering, monetization tracking, auto-draft reply (human approval)

> อย่า over-engineer ก่อน publish สม่ำเสมอ — phase 0 ให้ใช้งานจริงก่อน
